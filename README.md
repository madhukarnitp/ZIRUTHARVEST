# Zirut Harvest Landing Page

This project is a simple static landing page for **Zirut Harvest**, designed as a B2B partner portal for wholesale makhana and dry fruit enquiries.

## Project Structure

- `index.html` - Main single-page website with HTML, CSS, and JavaScript

## Features

- Responsive single-page layout
- Product showcase section
- Partnership tier section
- Testimonials and company information
- Contact and enquiry form
- Smooth scrolling navigation

## How to Use

1. Open `index.html` directly in your browser.
2. Or serve the folder with any simple local server if you prefer.

## Notes

- This project does not require any build tools or installation.
- External fonts and some product images are loaded from online sources.
